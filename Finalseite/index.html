emevero

Die effiziente Komplettlösung für Ihr Dokumentenmanagement

Emevero ist ein allgemein einsetzbares, intuitives und leistungsstarkes Dokumentenmanagementsystem mit schnellem Zugriff auf beliebige Dokumente.
Probieren Sie es aus.

Unsere Features

Versionierung

Sämtliche Dokumente werden versioniert abgelegt. Die Historie aller Änderungen ist somit einsehbar. Zusätzlich können Sie aber auch einen nativen Subversion Client verwenden.

Volltextsuche

Die Inhalte aller Dokumente werden in einer Datenbank indiziert. Geben Sie ihre gesuchten Begriffe ein und das passende Dokument wird in Sekundenschnelle gefunden.

Benachrichtigung von Nutzern

Für jedes Dokument gibt es die Möglichkeit bei einer Änderung ausgewählte Benutzer per E-Mail davon automatisch in Kenntnis zu setzen.

LDAP Anbindung

Für unser System ist keine eigene Benutzerverwaltung nötig. Binden Sie einfach ein bestehendes LDAP an. Auch die Authentifizierung von Benutzern erfolgt darüber mühelos.

Individuelle Rechtevergabe

Die Lese- und Schreibrechte sind vererbbar und können bis auf Dateiebene für jeden Benutzer individuell konfiguriert werden.

Einfache Benutzung

Wir legen besonderen Wert auf die Benutzerfreundlichkeit unserer Produkte. Deshalb halten wir die Benutzeroberfläche schlicht und übersichtlich. Intuitiv und schnell finden und bearbeiten Sie Ihre Dokumente.

Dateinamenvorgabe

Um Dateinamen konsistent vergeben zu können, kann mit Hilfe von regulären Ausdrücken eine Vorgabe erstellt werden, nach welchem Muster Dateien benannt werden sollen. Somit gewährleisten Sie einheitliche Namen unabhängig davon, wer sie erstellt.

Komplettservice

Wir richten für Sie alles ein und hosten das System auf einer eigenen Plattform. Bei Fragen und Problemen können Sie natürlich jederzeit zu arbeitsüblichen Zeiten unseren Telefonsupport nutzen.

Erweiterungen auf Kundenwunsch

Sie brauchen weitere Zusatzfeatures? Fast jeder Wunsch kann erfüllt werden. Sprechen Sie mit uns und wir werden eine Lösung finden.
